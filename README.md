# MBTA Transfer Helper

Next.js app with:
- centered tall app frame
- no overlapping panels
- Gemini station assist
- MBTA API live departure fetch
- local reliable route planning and map rendering

## Run

1. Copy `.env.example` to `.env.local`
2. Add your keys
3. Run:

```bash
npm install
npm run dev
```

Open http://localhost:3000
